A. 琪露诺的相交线段

题目描述

在二维平面中，给定一条与 xxx 轴平行的线段 ABABAB 和 一条与 yyy 轴平行的线段 CDCDCD。线段可以沿着坐标轴方向平移任意单位长度，记为一次操作，问需要几次操作使得两条线段相交。「YAC Round 5」琪露诺的相交线段

解题思路 基础计算几何

由题意两条线段分别与 xxx 轴和 yyy 轴平行，即 xA<xBx_A < x_BxA​<xB​ 且 yA=yBy_A = y_ByA​=yB​、xC=xDx_C = x_DxC​=xD​ 且 yC<yDy_C < y_DyC​<yD​。显然，只要同时满足 xA≤xC≤xBx_A \le x_C \le x_BxA​≤xC​≤xB​ 且 yC≤yA≤yDy_C \le y_A \le y_DyC​≤yA​≤yD​ 这两个条件时，两条线段相交。由于每次操作可以沿着坐标轴（上下左右）任意移动任何长度，故只需要判断初始情况下的两条线段满足条件的个数，即可得到答案。

B. 守矢斗牛三缺一

题目描述

给定五个不超过 101010 的正整数，判断是否可以凑出其中三个数字之和为 101010 的倍数。如果可以，输出剩下两个数字之和除以 101010 得到的余数 xxx（特别地，余数为 000，输出 101010）；否则输出 000。「YAC Round 5」守矢斗牛三缺一

解法一 递归（全排列）

我们可以递归枚举全排列，每次选取排列的前三个数，看这三个数之和是否是 101010 的倍数，如果是，则记录剩余两个数之和除以 101010 得到的答案。

复杂度为 O(n!)O(n!)O(n!)，但是由于 n=5n=5n=5，且数据组数不超过 10510^5105，大概需要 1.21.21.2 秒左右。特意没有卡这个做法。

解法二 枚举

我们可以反过来思考。只需要两两枚举数字，判断剩下三个数字之和是否是 101010 的倍数即可。故我们需要先求出五个数字之和，然后两两枚举记录答案。

解法三 数学 + 枚举

五个麻将数字之和 sumsumsum 除以 101010 得到余数 xxx，根据麻将斗牛的规则，我们可以发现，这五个数字最终要么就是 牛 xxx（特别的，余数为 000 时对应牛 101010），要么就是无牛。这也验证了一副麻将不会凑出其他的牛。

假设有 sum=a1+a2+…+a5sum = a_1 + a_2 + \ldots + a_5sum=a1​+a2​+…+a5​，x=sumx = sumx=sum mod\text{mod}mod 101010。若有 (ai+aj)(a_i + a_j)(ai​+aj​) mod\text{mod}mod 10=x10 = x10=x，那么剩余的数字之和 mod\text{mod}mod 101010 显然等于 000 。（取余运算的加法性质）

所以，我们只需要先求出五个数字之和除以 101010 的余数 xxx，然后两两枚举是否存在两个数字之和除以 101010 的余数正好是 xxx。如果存在，那么直接记录答案并退出；否则就是无牛。

C. 秘神摩多罗

题目描述

给定一个完全图，每个点有一个点权 aia_iai​，两个点之间的边权为 ai−2×aj+ca_i - 2 \times a_j + cai​−2×aj​+c，其中 ccc 为常数。每次询问一个点集，求包含点集的最短简单路径长度。（保证 ai−2×aj+ca_i - 2 \times a_j + cai​−2×aj​+c 非负）「YAC Round 5」秘神摩多罗

解题思路 贪心

这题和图论没有什么关系。

对于一个点集 SSS，其大小为 kkk，我们将点集中的点映射到 1 k1 ~ k1 k。我们可以先看一下，简单路径为 1→2→⋯→k1 \rightarrow 2 \rightarrow \dots \rightarrow k1→2→⋯→k 得到的路径长度：

(a1−2×a2+c)+(a2−2×a3+c)+…(ak−1−2×ak+c)(a_1 - 2 \times a_2 + c) + (a_2 - 2 \times a_3 + c) + \dots (a_{k-1} - 2 \times a_k + c) (a1​−2×a2​+c)+(a2​−2×a3​+c)+…(ak−1​−2×ak​+c)

显然，我们可以合并并消去一些项：

a1−a2−⋯−ak−1−2×ak+(k−1)×ca_1 - a_2 - \dots - a_{k-1} - 2 \times a_k + (k-1) \times c a1​−a2​−⋯−ak−1​−2×ak​+(k−1)×c

化简可得：

2×a1−ak+∑i=1k(−ai)+(k−1)×c2 \times a_1 - a_k + \sum_{i=1}^k (-a_i) + (k-1) \times c 2×a1​−ak​+i=1∑k​(−ai​)+(k−1)×c

故我们可以发现，简单路径的长度 只和起点和终点有关。

而我们需要让长度尽可能小，显然，我们只需要最小化 a1a_1a1​，并最大化 aka_kak​ 即可。故对于每次询问的点集，我们只需要记录点集中的最大值和最小值。对于 ∑i=1k(−ai)\sum_{i=1}^k (-a_i)∑i=1k​(−ai​) ，只需要在记录最大和最小值的同时累加一下即可。（注意开 long long）

D. 异或操作真是太棒啦

题目描述

两种操作，第一种为修改第 kkk 个数为 xxx；第二种为询问一个区间 [l,r][l, r][l,r]，求出区间内 所有子区间异或和的异或和。https://www.luogu.com.cn/problem/T426567

解题思路 树状数组 + 位运算

对于区间 [l,r][l, r][l,r]，根据按位异或的交换律以及两个相等的数异或抵消为 0 的性质，我们可以发现：

al⊕al+1⊕⋯⊕ara_l \oplus a_{l+1} \oplus \dots \oplus a_ral​⊕al+1​⊕⋯⊕ar​ 抵消 (al⊕al+1⊕⋯⊕ar)(a_l \oplus a_{l+1} \oplus \dots \oplus a_r)(al​⊕al+1​⊕⋯⊕ar​)

(al⊕al+1)⊕(al+1⊕al+2)⊕⋯⊕(ar−1⊕ar)(a_l \oplus a_{l+1}) \oplus (a_{l+1} \oplus a_{l+2}) \oplus \dots \oplus (a_{r-1} \oplus a_{r})(al​⊕al+1​)⊕(al+1​⊕al+2​)⊕⋯⊕(ar−1​⊕ar​) 抵消 (al⊕al+1⊕al+2⊕⋯⊕ar−1)⊕(al+1⊕al+2⊕⋯⊕ar−1⊕ar)(a_l \oplus a_{l+1} \oplus a_{l+2} \oplus \dots \oplus a_{r-1}) \oplus (a_{l+1} \oplus a_{l+2} \oplus \dots \oplus a_{r-1} \oplus a_{r})(al​⊕al+1​⊕al+2​⊕⋯⊕ar−1​)⊕(al+1​⊕al+2​⊕⋯⊕ar−1​⊕ar​)

…\dots… 以此类推

因此，当区间长度 r−l+1r - l + 1r−l+1 为偶数时，所有项全部抵消为 000；

而当区间长度 r−l+1r - l + 1r−l+1 为奇数时，记 len=r−l+1len = r - l + 1len=r−l+1，则最后剩下的是：

(al⊕al+1⊕⋯⊕alen+12)⊕(al+1⊕al+2⊕⋯⊕alen+12⊕alen+12+1)⊕⋯⊕(alen+12⊕alen+12+1⊕⋯⊕ar)(a_l \oplus a_{l+1} \oplus \dots \oplus a_{\frac{len + 1}{2}}) \oplus (a_{l+1} \oplus a_{l+2} \oplus \dots \oplus a_{\frac{len + 1}{2}} \oplus a_{\frac{len + 1}{2} + 1}) \oplus \dots \oplus (a_{\frac{len + 1}{2}} \oplus a_{\frac{len + 1}{2} + 1} \oplus \dots \oplus a_{r}) (al​⊕al+1​⊕⋯⊕a2len+1​​)⊕(al+1​⊕al+2​⊕⋯⊕a2len+1​​⊕a2len+1​+1​)⊕⋯⊕(a2len+1​​⊕a2len+1​+1​⊕⋯⊕ar​)

可以发现，区间内 第奇数个元素，经过抵消后最后还剩下一个，而 第偶数个元素，全部抵消。

故对于每次询问的区间，我们只需要输出区间 [l,r][l, r][l,r] 内所有第奇数个元素的异或和即可。

由于涉及到单点修改，我们可以用开两个树状数组，为 tr0tr_0tr0​ 和 tr1tr_1tr1​，分别维护数组中偶数位的异或和 与 奇数位的异或和。

对于区间 [l,r][l,r][l,r]，如果区间长度为偶数，那么就直接输出 000；

如果区间长度为奇数，若 rrr 为奇数，那么只需要取出 tr1tr_1tr1​ 中对应的区间异或和即可；若 rrr 为偶数，那么取出 tr0tr_0tr0​ 中对应的区间异或和即可。

F. The World 世界

题目描述

给定一个 NNN 个点和 MMM 条有向边的图，最多可以进行 KKK 次让一条边权值变为原先相反数的操作，问 111 到 NNN 的最短距离是多少。「YAC Round 5」The World 世界

解题思路 倍增Floyd（矩阵快速幂）

我们可以用动态规划来解决这个图论问题（floyd）。首先我们需要预处理得到不使用魔法以及最多使用一次魔法得到的最短路，用 d[i][j][p]d[i][j][p]d[i][j][p] 表示从 iii 到 jjj 最多使用 ppp 次魔法的最短路长度，状态转移如下：

  • 不使用魔法

    d[i][j][0]=min⁡(d[i][j][0],d[i][k][0]+d[k][j][0])d[i][j][0] = \min(d[i][j][0], d[i][k][0] + d[k][j][0]) d[i][j][0]=min(d[i][j][0],d[i][k][0]+d[k][j][0])

    就是最简单的 Floyd\text{Floyd}Floyd 模板。

  • 最多使用一次魔法

    不使用魔法有可能更优

    d[i][j][1]=min⁡(d[i][j][1],d[i][j][0])d[i][j][1] = \min(d[i][j][1], d[i][j][0]) d[i][j][1]=min(d[i][j][1],d[i][j][0])

    使用一次魔法会更优

    d[i][j][1]=min⁡(d[i][j][1],d[i][u][0]+d[v][j][0]−w)d[i][j][1] = \min(d[i][j][1], d[i][u][0] + d[v][j][0] - w) d[i][j][1]=min(d[i][j][1],d[i][u][0]+d[v][j][0]−w)

    其中 (u,v,w)(u, v, w)(u,v,w) 为枚举的使用魔法的边。

然后，我们就可以由这两个初始状态得到最多使用 KKK 次魔法的最短路。状态转移如下：

d[i][j][p]=min⁡(d[i][j][p],d[i][k][p−1]+d[k][j][1])d[i][j][p] = \min(d[i][j][p], d[i][k][p - 1] + d[k][j][1]) d[i][j][p]=min(d[i][j][p],d[i][k][p−1]+d[k][j][1])

用最多使用 p−1p - 1p−1 次魔法和最多使用一次魔法转移得到最多使用 ppp 次魔法得到的最短路。在求解最多使用 ppp 次时，显然最多使用 p−1p-1p−1 次魔法是先前已经求出来的。
